SETUP                                               *neorg-flashcards-setup*

Install the plugin with your plugin manager, then configure at least one
language:
>
  local presets = require("neorg_flashcards.presets")

  require("neorg_flashcards").setup({
    flashcards_dir = vim.fn.expand("~/notes/flashcards"),
    default_file = vim.fn.expand("~/notes/flashcards/cards.norg"),
    default_kind = "japanese",
    languages = presets.only("japanese"),
  })
<

Use `presets.only("chinese")` for Chinese, or pass a custom `languages` table.

Neorg is optional. Card storage and collection scans always use `.norg` files,
but the plugin reads those files directly. Without Neorg they are ordinary
plain-text buffers. Files ending in `.md` or `.txt` are not collected.

==============================================================================
COMMANDS                                         *neorg-flashcards-commands*

                                                        *:NeorgFlashcardOpen*
:NeorgFlashcardOpen
        Create or open the configured default flashcard file. Missing parent
        directories are created.

                                                         *:NeorgFlashcardAdd*
:NeorgFlashcardAdd [kind]
        Prompt for a new card. Adds to the current `.norg` file, or opens
        `default_file` from any other buffer. Uses `default_kind` when [kind]
        is omitted.

                                                      *:NeorgFlashcardReview*
:NeorgFlashcardReview
        Review all valid cards under `flashcards_dir`.

                                                  *:NeorgFlashcardReviewFile*
:NeorgFlashcardReviewFile
        Review valid cards in the current buffer.

                                                   *:NeorgFlashcardReviewTag*
:NeorgFlashcardReviewTag [tag]
        Review cards with a matching `tags:` value.

                                                 *:NeorgFlashcardReviewScore*
:NeorgFlashcardReviewScore [bad|mid|good|new]
        Review cards by saved score.

                                                    *:NeorgFlashcardValidate*
:NeorgFlashcardValidate
        Validate flashcard blocks in the current buffer.

                                                        *:NeorgFlashcardHelp*
:NeorgFlashcardHelp
        Open the short in-editor guide.

==============================================================================
KEYMAPS                                         *neorg-flashcards-keymaps*

The plugin defines commands but no global keymaps. Review keys are local to
the review popup. Example global mappings:
>
  vim.keymap.set("n", "<leader>nci", "<cmd>NeorgFlashcardAdd<CR>",
    { desc = "Add flashcard" })
  vim.keymap.set("n", "<leader>ncr", "<cmd>NeorgFlashcardReview<CR>",
    { desc = "Review all flashcards" })
  vim.keymap.set("n", "<leader>ncf", "<cmd>NeorgFlashcardReviewFile<CR>",
    { desc = "Review current flashcard file" })
  vim.keymap.set("n", "<leader>nct", "<cmd>NeorgFlashcardReviewTag<CR>",
    { desc = "Review flashcards by tag" })
<

Review keys:

  Space / Enter   reveal answer, then advance
  1               save score 1 and advance
  2               save score 2 and advance
  3               save score 3 and advance
  n               next card
  p               previous card
  e               edit source card
  q               close review

==============================================================================
CHAPTERS AND COLLECTIONS                    *neorg-flashcards-collections*

Keep one chapter per `.norg` file under `flashcards_dir`. Open a chapter with
Neovim, add cards with `:NeorgFlashcardAdd`, and study only that chapter with
`:NeorgFlashcardReviewFile`. `:NeorgFlashcardReview` recursively combines all
chapter files under `flashcards_dir`.

Use tags for topics that span chapters. For example, add `tags: grammar` to
cards in several files and run `:NeorgFlashcardReviewTag grammar`.

`:NeorgFlashcardOpen` opens `default_file`; it is not a file picker. Keep the
default file under `flashcards_dir` when it should be part of collection
reviews.

==============================================================================
CARD FORMAT                                   *neorg-flashcards-card-format*
>
  @flashcard japanese
  japanese: 勉強
  reading: べんきょう
  english: study
  notes: noun / suru verb
  tags: jlpt vocab
  score: 2
  reviewed: 2026-07-01
  @end
<

`score:` and `reviewed:` are updated by the review UI when rating with 1, 2,
or 3.
